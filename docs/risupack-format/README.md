# RisuPack Format Reference

이 폴더는 `RisuPack/docs/format/`의 참조본이다.

`RisuAI-Asset-Extractor`에서 아래 문서를 포맷 기준으로 사용한다.

- `charx.md`
- `risum.md`
- `risup.md`
- `gotchas.md`

운영 원칙은 아래와 같다.

1. 포맷 기준은 `RisuPack` 문서를 우선한다.
2. 포맷 버그나 해석 차이가 발견되면 `RisuPack` 쪽을 먼저 수정한다.
3. 그 다음 이 폴더의 참조본과 `RisuAI-Asset-Extractor` 구현을 맞춘다.
